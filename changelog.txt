[B]Retrospect v5.2.6 - Changelog - 2020-06-09[/B]

This new version of Retrospect addresses a number of channel bugs. Besides this the behavior of the bitrate limiting was changed and by default no limit is set. Users can still change this via the add-on settings. There was also a fix for changed Kodi Matrix Python API.

[B]Framework related[/B]
* Fixed: cleaning up cache and cookies sometimes did not clean cookies

[B]GUI/Settings/Language related[/B]
* Fixed: Regression bug that caused the description prefixes to not be on a new line
* Changed: sorting code optimized and added an sort by 'episode' option that users can select per folder

[B]Channel related[/B]
* Fixed: VRT.nu could not log in any more (Fixes #1400)
* Removed: SBS6, SBS9, Veronica, NET5 as they all moved to kijk.nl (Fixes #1401)
* Fixed: Kijk.nl subtitles were missing (Fixes #1402)
* Fixed: Too many episodes show as geo restricted in NPO
