[B]Retrospect v5.2.x - Changelog - 2020-05-07[/B]

This new version of Retrospect fixes some channel related issues and some minor core refactoring.

[B]Framework related[/B]
* Changed: the full channel ID's are now added to the url's
* Changed: load channels based on their full ID's instead of the channelindex.json
* Added: option to limit certain settings to only some channel in a channel set
* Added: option to generate a channel guid for a setting in a channel set

[B]GUI/Settings/Language related[/B]
_None_

[B]Channel related[/B]
* Added: option to specify a login for Dplay.se (Fixes #1364)
